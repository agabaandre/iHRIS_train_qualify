Gerenciar Pessoas

Adicionar Pessoa

Para procurar uma pessoa no banco de dados, seja um empregado ou um candidato ao emprego, adicione um registro para essa pessoa. Certas informações são necessárias para um novo registro. Uma vez que o registro foi gerado, opções adicionais para adicionar dados sobre a pessoa estarão disponíveis. Um empregado do RH ou um gerente do RH pode adicionar uma pessoa para o sistema. Rever Candidatos

Se uma ou mais aplicações forem gravadas para um cargo aberto, um funcionário, um gerente ou um Gerente Executivo pode revisar as aplicações e gravar notas durante o processo de contratação. Clique em Revisar Candidatos para rever todas as aplicações para um cargo aberto
