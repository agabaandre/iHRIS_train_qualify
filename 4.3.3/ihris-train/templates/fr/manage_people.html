Gérer les personnes

Ajouter une personne

Pour suivre une personne dans la base de données, ajouter un enregistrement pour cette personne. Certaines informations sont obligatoires pour commencer un nouvel enregistrement. Une fois l’enregistrement généré, des options supplémentaires pour ajouter des données sur la personne deviendront disponibles.

Ajouter du personnel

Cela permet de créer un nouveau Conférencier, Chef d’établissement, Chef d’établissement adjoint, Chef de clinique, Administrateur académique, Coordinateur de niveau, etc.
