Добавить личность

Чтобы отследить человека в базе данных, служащего или соискателя, добавьте запись для этого человека. Для того, чтобы внести новую запись, необходимо ввести некоторую информацию. Как только запись будет создана, будут доступны дополнительные опции добавления данных для этого человека. Служащий отдела кадров или управляющий кадрами могут добавлять новые записи в систему.
