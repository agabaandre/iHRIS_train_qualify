Pamahalaan ang mga Tao

Magdagdag ng Tao

Upang masubaybayan ang isang tao sa loob ng kalipunan ng tao, kung empleyado man o isang aplikante para sa hanapbuhay, magdagdag ng isang rekord para sa taong iyan. Kailangan ang partikular na kabatiran upang makapagsimula ng isang bagong rekord. Kapag nalikha na ang isang rekord, ang karagdagang mga mapagpipilian para sa pagdaragdag ng dato hinggil sa tao ay magiging nakukuha. Ang isang taong tauhan ng Kuhanan ng Tao o isang Tagapamahala ng Kuhanan ng Tao ay makapagdaragdag ng isang bagong tao sa sistema. Upang masubaybayan ang isang tao sa loob ng kalipunan ng dato, magdagdag ng isang rekord para sa taong iyon. Kailangan ang partikular na kabatiran upang makapagsimula ng isang rekord. Kapag nalikha na ang isang rekord, makakakuha ng karagdagang mga mapagpipilian para sa pagdaragdag ng dato tungkol sa tao. Ang isang taong tauhan ng Kuhanan ng Tao o isang Tagapamahala ng Kuhanan ng Tao ang makapagdaragdag ng isang tao sa sistema.

Suriin ang mga Aplikante

Kapag naitala na ang isa o mahigit pang mga aplikasyon para sa isang bukas na posisyon, isang taong Tauhan ng Tagakuha ng Tao, Tagapamahala ng Kuhanan ng Tao, o Tagapamahalang Tagapagpatupad ay maaaring makapagsuri ng mga aplikasyon at makapagtatala ng mga tala sa kahabaan ng pagkuha ng tauhan. Pindutin ang Suriin ang mga Aplikante upang suriin ang lahat ng mga aplikasyon para sa anumang bukas na posisyon.
